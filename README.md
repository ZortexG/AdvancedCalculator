## Advanced Calculator

**Current State: Still Under Development**

## Description

Advanced Calculator is an all-around calculator that has all the necessary features. It uses C++ as a base and wxWidgets for the GUI Library.

## Requirements
- Windows (Visual Studio)
- Visual Studio 2022 (any VS that supports C++14)
- wxWidgets
- C++14 compiler support

## Features
For now, it is a basic calculator, but more features are coming soon.

## Planned Features
- Root and Sqrt features
- A Quick Select algebra and trigonometry function
- QoL and UI Enhancements
- And much more
